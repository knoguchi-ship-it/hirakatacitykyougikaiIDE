# リリース記録 v241（2026-04-19）

v241 では、運用上の正本 URL を `https://script.google.com/a/macros/hcm-n.org/s/AKfycbxyuUXgK1oHUDMahQjluiL-gcrMK0qV0FWLFYaYBqGxlRSg9NhvmbyQRyf0dvaqg7Zp/exec` に統一した。

## 変更内容

### 1. 会員マイページ URL の正本を単一化
- `backend/Code.gs` の `MEMBER_PORTAL_URL` を、public fixed deployment の base URL に変更。
- `doGet()` は `app` 未指定時に `member` を返すため、この URL を会員/管理者ログインの正規入口として扱う。
- 公開ポータルは同じ base URL に `?app=public` を付けた派生 URL として扱う。

### 2. 文書の入口を運用実態に合わせて更新
- `docs/02_ARCHITECTURE.md` の URL 構成を単一運用入口前提に更新。
