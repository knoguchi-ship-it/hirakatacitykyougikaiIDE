## 4. Security Basis

2026-04-29 時点の一次ソース確認:

- Google Apps Script `google.script.run`: HTML-service pages can call server-side Apps Script functions asynchronously.
- Google Apps Script private functions: names ending with `_` cannot be called by `google.script.run` and are invisible to the client.
- Google Apps Script Web Apps: web apps can run as the deploying user, so public callable server functions must be minimized.
- OWASP WSTG latest authorization testing: bypassing authorization schema, privilege escalation, IDOR, and OAuth weakness are explicit test areas.
- OWASP ASVS latest stable and NIST SP 800-63-4 were used as baseline references for least privilege and boundary enforcement.
