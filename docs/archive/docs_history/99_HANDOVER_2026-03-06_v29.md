## 3. 本セッション（v28→v29）での修正内容

### 3-1. トグルスイッチ onClick バグ修正（`src/components/TrainingManagement.tsx`）

v28 でフィールド設定トグルが動作しない不具合を修正しました。

**原因**: `onClick` ハンドラが内側の `<span>`（トグルピル）にのみ付与されており、
ラベルテキストのエリアをクリックしても反応しなかった。

**修正内容**: `onClick` を外側の `<label>` 要素に移動。

```tsx
// 修正前（v28 — ラベルテキストをクリックしても反応しない）
<label key={key} className="flex items-center gap-2 cursor-pointer select-none group">
  <span
    onClick={() => toggleField(key)}
    className={`relative inline-flex h-5 w-9 ...`}
  >

// 修正後（v29 — ラベル全体をクリックでトグル反応）
<label key={key} className="flex items-center gap-2 cursor-pointer select-none group" onClick={() => toggleField(key)}>
  <span
    className={`relative inline-flex h-5 w-9 ...`}
  >
```

## 6. 重要な運用制約

| 制約 | 詳細 |
|---|---|
| ビルドコマンド | **`npm run build:gas`** を使用（`npm run build` では backend/index.html が更新されない） |
| Web App デプロイ | `clasp deploy` は **API Executable** になる。Web App は **Apps Script UI** からのみ更新可能 |
| clasp push | `npx clasp push --force` が必要 |
| ファイルアップロード | base64 で渡し Drive 保存。上限 5MB |
| Drive フォルダ | 「研修案内状」フォルダ（自動作成）。共有設定: リンクを知っている全員が閲覧可 |
