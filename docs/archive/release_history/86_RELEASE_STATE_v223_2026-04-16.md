## このリリースで対応した課題

### 公開ポータル 入会前案内のダイアログ化と定款リンク追加

**対象ファイル**: `src/components/application/MemberApplicationForm.tsx`

#### 変更内容
- 常設表示していた「事務局からのお願い」カードを廃止
- 見出し
  - `事務局からのお願い（ご入会にあたって）`
  - `入会手続きの前に、重要事項をご確認ください。`
  の横に配置したボタンからモーダルダイアログを開く方式へ変更
- ダイアログ内では、重要事項を4つの要点カードで確認できる構成に整理
- 「変更・退会の手続き」カード内に、協議会ホームページへの導線を配置
  - `https://sites.google.com/view/starhirakata/%E5%85%A5%E4%BC%9A%E9%80%80%E4%BC%9A?authuser=0`
- ダイアログ内に定款確認導線を追加
  - `https://sites.google.com/view/starhirakata/incorporation`
- 全文ブロックは廃止
- ダイアログ内で確認チェックまたは「内容を確認して閉じる」を行うまで、会員種別選択は不可のまま維持
- `Escape` キーでダイアログを閉じられるようにした

#### UI / UX 意図
- 初期画面の情報量を抑えつつ、重要事項は明確に別レイヤーで確認させる
- ホームページ導線と定款導線を「確認のための関連リンク」として同じ文脈に統合する
- 一覧的に読みやすい要点カードと、閉じた後に残る `確認済み / 未確認` 状態表示で進行状態を分かりやすくする
