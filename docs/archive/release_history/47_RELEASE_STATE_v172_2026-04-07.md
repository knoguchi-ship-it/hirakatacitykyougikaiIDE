- 事業所会員の代表者情報は `REPRESENTATIVE` 本人のみ編集可能。
- 代表者情報を編集できない利用者には、入力欄をグレーアウトして閲覧専用であることを明示する。
- 事業所会員の `ADMIN` は代表者情報を閲覧のみとし、代表者行の状態・権限は会員マイページから変更不可。

- 事業所会員の入会日はグレーアウトした表示専用フィールドで表示する。
- `WITHDRAWAL_SCHEDULED` の場合は、退会予定であることと退会予定日を表示する。
