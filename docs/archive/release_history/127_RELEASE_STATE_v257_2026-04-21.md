# リリース状態記録: v257

日付: 2026-04-21
GAS version: 統合@255 / 会員 split@8 / 管理者 split@12

## 概要

管理者向けのシステム設定画面を情報設計から再編し、長大な単一フォームを、概要ヘッダ、クイックジャンプ、アコーディオン式セクション、sticky 保存バーを持つ構成へ刷新した。既存の設定キー、state、保存 API は維持し、運用導線と可読性を改善した。

## 変更内容

### システム設定画面

- 画面上部にページ概要、保存状態、設定読込状態、会員個別上限の状態カードを追加
- セクションへのアンカーリンクを追加
- `details/summary` ベースの `AdminSettingsSection` を導入
- 設定画面を以下の 5 セクションへ再編
  - 基本設定
  - 帳票・一括メール
  - 公開ポータル
  - 入会通知メール
  - 事業所ごとの個別上限
- 下部アクションを sticky 保存バーへ変更し、未保存状態を常時表示

### 実装上の注意

- 保存 payload と backend API は変更していない
- 既存の `T_システム設定` キー構造は維持
- UI の情報設計変更が主であり、設定値の意味や保存先は従来互換
