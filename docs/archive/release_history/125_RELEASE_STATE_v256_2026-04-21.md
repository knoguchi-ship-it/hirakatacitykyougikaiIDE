# リリース状態記録: v256

日付: 2026-04-21
GAS version: 統合@254 / 会員 split@7 / 管理者 split@11

## 概要

公開ポータルの入会申込完了画面に表示される `今後のご案内` と `ログイン情報` を、システム設定画面から表示/非表示および本文変更できるようにした。文言は `T_システム設定` に外出しし、画面側ではプレーンテキストとして安全に描画する。

## 変更内容

### システム設定

- `今後のご案内` ブロックの表示 ON/OFF を追加
- `今後のご案内` 本文をメール送信 ON/OFF 別に編集可能化
- `ログイン情報` ブロックの表示 ON/OFF を追加
- `ログイン情報` 補足本文をメール送信 ON/OFF 別に編集可能化
- 既存の `ログイン情報を画面に表示する` トグルは継続利用

### 追加キー

- `PUBLIC_PORTAL_COMPLETION_GUIDANCE_VISIBLE`
- `PUBLIC_PORTAL_COMPLETION_GUIDANCE_BODY_WHEN_CREDENTIAL_SENT`
- `PUBLIC_PORTAL_COMPLETION_GUIDANCE_BODY_WHEN_CREDENTIAL_NOT_SENT`
- `PUBLIC_PORTAL_COMPLETION_LOGIN_INFO_BLOCK_VISIBLE`
- `PUBLIC_PORTAL_COMPLETION_LOGIN_INFO_BODY_WHEN_CREDENTIAL_SENT`
- `PUBLIC_PORTAL_COMPLETION_LOGIN_INFO_BODY_WHEN_CREDENTIAL_NOT_SENT`

### 実装上の注意

- 文言は HTML として解釈せずプレーンテキストのまま描画
- 改行は `whiteSpace: 'pre-line'` で保持
- 旧キー `PUBLIC_PORTAL_COMPLETION_NO_CREDENTIAL_NOTICE` / `PUBLIC_PORTAL_COMPLETION_CREDENTIAL_NOTICE` は後方互換のため維持
- 新キー未設定時は旧キーからフォールバック生成
