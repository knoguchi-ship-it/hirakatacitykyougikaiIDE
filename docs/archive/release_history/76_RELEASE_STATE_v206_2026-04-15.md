## Summary

`T_会員` に `勤務先住所2`（事業所・勤務先の建物名）と `自宅住所2`（自宅の建物名）列を追加。
デジタル庁アドレスベースレジストリ / ISO 19160-4 準拠の2段階住所構造（番地 + 建物名）。

## 本番DB適用手順

```bash
npx clasp run addAddressLine2Columns
```

- 実行後: `勤務先住所2` / `自宅住所2` 列が `T_会員` シートに挿入される
- 既存行の新列は全行空白（会員が次回編集時に入力）
- 冪等: 列が既に存在する場合はスキップ

> **注意**: デプロイ前にこの関数を実行すると、本番DBと旧コードの列数が一時的に不一致になるため、  
> 必ず **clasp push → version → redeploy 完了後** に実行すること。
