# Release State: v252 (2026-04-21)

## 概要

入会完了画面に表示される「ログイン情報未送信時の案内文」を管理設定から変更可能にした。
会員マイページ公開前の運用状態に合わせた文言をシステム設定から自由に設定できる。

---

## 変更内容

### 背景

`credentialEmailEnabled=false`（ログイン情報メール未送信）の間、入会完了画面には
「ログイン情報メールは現在送信していません。会員ページの公開準備後にご案内します。」
というハードコードされた文言が表示されていた。運用フェーズに合わせて管理者が変更できなかった。

### 新規 DB キー: `PUBLIC_PORTAL_COMPLETION_NO_CREDENTIAL_NOTICE`

- `credentialEmailEnabled=false` 時に入会完了画面の「今後のご案内」ブロックおよびログイン情報カードに表示する案内文
- デフォルト値: `ログイン情報メールは現在送信していません。会員ページの公開準備後にご案内します。`
- 空にするとデフォルト値にフォールバック

### 管理設定画面

「ログイン情報送信設定」セクションの既存トグル群の下に追加:
- `textarea`: 案内文入力（2行・リサイズ可）
- 「デフォルトに戻す」ボタン付き

### MemberApplicationForm の変更

- `completionNoCredentialNotice` prop を追加（デフォルト = 上記デフォルト値）
- `loginInfoNotice`（今後のご案内ブロック）と非表示時のカード文言が prop 値で上書きされる
- `credentialEmailEnabled=true` 時のメッセージ（送信済み確認文）は変更なし
