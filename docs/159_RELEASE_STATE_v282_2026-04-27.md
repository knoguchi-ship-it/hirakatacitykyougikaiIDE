## 変更内容

### split action handler pruning

**目的**: v281 の action registry 物理削減を一段進め、`processApiRequest` 内の境界外 action handler 分岐も split 生成物から削除する。

**修正内容**:
- `scripts/build-member-gas.mjs`
  - member allowlist 外の `if (action === '...')` handler block を生成時に削除。
  - member 生成物の `processApiRequest` は member action のみ保持。
- `scripts/build-admin-gas.mjs`
  - admin allowlist 外の `if (action === '...')` handler block を生成時に削除。
  - admin 生成物の `processApiRequest` は admin action のみ保持。

## 影響範囲

直接影響:

- `gas/member/Code.gs`
- `gas/admin/Code.gs`
- `scripts/build-member-gas.mjs`
- `scripts/build-admin-gas.mjs`

仕様上の影響:

- member split から admin / public action handler 分岐を削除。
- admin split から member self-service / public action handler 分岐を削除。
- 統合 / 公開 deployment は変更せず `@283` を維持。
- 関数本体そのものは削除していないため、shared helper 誤削除による破損リスクは今回発生させていない。
