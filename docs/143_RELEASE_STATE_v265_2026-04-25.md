## 変更内容

### 1. 事業所会員 入会時メール 代表者/メンバー分離

旧: 全職員（代表者・メンバー問わず）に同一テンプレートでメール送信  
新: 代表者・メンバーで別テンプレート・別ON/OFFで送信制御

| 設定キー | 説明 |
|---|---|
| `BIZ_REP_EMAIL_ENABLED` | 事業所入会時：代表者メール ON/OFF |
| `BIZ_REP_EMAIL_SUBJECT` | 代表者メール 件名 |
| `BIZ_REP_EMAIL_BODY` | 代表者メール 本文 |
| `BIZ_STAFF_EMAIL_ENABLED` | 事業所入会時：メンバーメール ON/OFF |
| `BIZ_STAFF_EMAIL_SUBJECT` | メンバーメール 件名 |
| `BIZ_STAFF_EMAIL_BODY` | メンバーメール 本文 |

全体フラグ（`CREDENTIAL_EMAIL_ENABLED`）が最優先。OFF の場合は個別設定に関わらず全停止。

### 2. 職員追加申請 承認時メール（新規）

管理者が変更申請コンソールで職員追加申請を承認した際に自動送信されるメール2種。

| 設定キー | 説明 |
|---|---|
| `STAFF_ADD_STAFF_EMAIL_ENABLED` | 追加職員へのメール ON/OFF |
| `STAFF_ADD_STAFF_EMAIL_SUBJECT` | 追加職員メール 件名 |
| `STAFF_ADD_STAFF_EMAIL_BODY` | 追加職員メール 本文 |
| `STAFF_ADD_REP_EMAIL_ENABLED` | 代表者通知メール ON/OFF |
| `STAFF_ADD_REP_EMAIL_SUBJECT` | 代表者通知メール 件名 |
| `STAFF_ADD_REP_EMAIL_BODY` | 代表者通知メール 本文 |

### 3. テンプレート変数

| 変数 | 使える場面 |
|---|---|
| `{{氏名}}` | 全メール |
| `{{ログインID}}` | 代表者・メンバー・追加職員メール |
| `{{パスワード}}` | 代表者・メンバー・追加職員メール |
| `{{会員マイページURL}}` | 全メール |
| `{{事業所名}}` | 全メール |
| `{{追加職員氏名}}` | 代表者通知メールのみ（複数時は読点区切り） |

### 4. バックエンド

- `renderBizEmailTemplate_(template, vars)` — `{{変数}}` 置換ヘルパー
- `getBizEmailSettings_(ss)` — 事業所メール設定一括取得ヘルパー
- `submitMemberApplication_` — 事業所会員の代表者/メンバー分岐メール送信
- `approveAdminChangeRequest_` — STAFF_ADD 承認時に追加職員メール + 代表者通知送信
- `ensureSystemSettingsRows_` — 12設定キーのデフォルト初期化

### 5. 管理設定UI

- 管理者ポータル → システム設定 → 「事業所会員メール設定」セクション追加（v265時点）
  ※ v266 で「入会・登録メール設定」に統合・再設計
