TypeScript 入門

TypeScript は JavaScript に「型」の考え方を加えたものです。 プログラムを実行する前に、データの形の間違いを見つけやすくするために使います。

JavaScript

JavaScript だけの場合

自由に書ける一方で、実行するまで間違いに気づきにくいことがあります。

const user = { name: "Yamada" };
console.log(user.email); // 実行時まで気づきにくい
TypeScript

TypeScript を使う場合

「そのデータに何が入っているか」を先に決めるので、誤りを早めに見つけやすくなります。

type User = {
  name: string;
  email: string;
};

const user: User = { name: "Yamada" }; // ここで不足を検出しやすい
