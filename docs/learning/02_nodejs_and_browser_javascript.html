Node.js とブラウザ JavaScript の違い

JavaScript という言語自体は同じでも、動く場所が違うと使える機能が変わります。 この違いを理解すると、「なぜローカルでは動かないのか」「なぜブラウザでだけ使えるのか」が分かりやすくなります。

ブラウザ側

ブラウザ JavaScript

  • 画面を表示する
  • ボタンや入力欄を扱う
  • window や document が使える
  • HTML や CSS と連携する
Node.js 側

Node.js 上の JavaScript

  • ファイルを読む・書く
  • コマンドを実行する
  • ビルドや型チェックを実行する
  • 通常は document や window は使えない

ブラウザで使いやすいもの

document.getElementById()

alert()

localStorage

≠

Node.js で使いやすいもの

fs.readFileSync()

process.env

child_process.execSync()
