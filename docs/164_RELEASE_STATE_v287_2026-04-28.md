## 変更内容

### member/admin physical function pruning

目的: URL と既存機能を維持したまま、member/admin split の生成済み `Code.gs` から境界外関数を物理削除する。

修正内容:

- `scripts/build-member-gas.mjs` に到達不能トップレベル関数削除を再導入。
- `scripts/build-admin-gas.mjs` に到達不能トップレベル関数削除を導入。
- v283 の再発防止として、削除対象関数を参照するトップレベル文も同時に削除。
- member 生成時は admin 権限分岐 `if (requiredPerms)` を削除。
- admin 生成時は member セッション分岐を削除。
- member 生成物から 452 関数と 24 トップレベル文を削除。
- admin 生成物から 326 関数と 20 トップレベル文を削除して `@47` に同期したが、実ブラウザでホワイトアウトが発生したため `@46` へロールバック。
- 統合/公開 deployment は URL 維持条件のため今回は変更なし。
