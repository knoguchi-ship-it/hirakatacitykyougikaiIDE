固定 deployment: member `@191` / public `@191`

## 実施内容: カスタム deflate-raw+base64 HTML 圧縮

### 背景・経緯

- v188: `vite-plugin-singlefile` + Terser。member 522 kB（raw）。
- v189: `vite-plugin-singlefile-compression` を試みたが**画面真っ白**で失敗。
  - 原因: プラグインが `import(URL.createObjectURL(blob))` を使用。
  - GAS CSP（`script.googleusercontent.com`）は `blob:` スキームの動的 import をブロックする。
- v190: `vite-plugin-singlefile` に復旧。
- v191: GAS CSP 互換の方式でカスタム圧縮を実装。

### 実装方針

| 比較項目 | v189（失敗）| v191（本実装） |
|---|---|---|
| 圧縮アルゴリズム | deflate-raw + Base128 | deflate-raw + Base64 |
| 実行方式 | `import(blob:URL)` | `new Function(text)()` |
| GAS CSP | ブロックされる | 許可（unsafe-eval 許可済み） |
| バンドル前提 | import/export なし | import/export なし ✓ |

### 変更ファイル

#### `scripts/compress-html.mjs` (新規)

- `dist/index.html` と `dist-public/index_public.html` の `<script type="module" crossorigin>` を圧縮
- Node.js `zlib.deflateRawSync()` で deflate-raw 圧縮（Z_BEST_COMPRESSION）
- Base64 エンコード後、`<script id="__app_data__" type="application/octet-stream">` に格納
- 実行時デコンプレッサー:
  ```javascript
  (async function(){
    var d=document.getElementById('__app_data__').textContent.trim();
    var b=atob(d),n=new Uint8Array(b.length);
    for(var i=0;i<b.length;i++)n[i]=b.charCodeAt(i);
    var t=await new Response(new Response(n).body.pipeThrough(new DecompressionStream('deflate-raw'))).text();
    new Function(t)();
  })();
  ```

#### `scripts/build-gas.mjs` (変更)

- `npx vite build`（両ポータル）の後に `node scripts/compress-html.mjs` を実行するステップを追加
