## Summary

宛名リスト Excel 出力コンソール（MailingListExport）を追加。
事業所会員・個人会員・賛助会員の3シート + 住所不備シートの構成で、
広報誌発送・お知らせ発送の2フィルタモードに対応。

## Changes

### GAS backend (`backend/Code.gs`)

**`generateMailingListExcel_(payload)`** — 新規追加

| 項目 | 内容 |
|---|---|
| filterType | `'KOHOUSHI'`（広報誌発送）または `'OSHIRASE'`（お知らせ発送） |
| 対象会員 | ACTIVE + WITHDRAWAL_SCHEDULED（退会予定は含むがステータスは出力しない） |
| OSHIRASE フィルタ | 事業所会員: 全員 / 個人・賛助: 発送方法コード = 'POST' の方のみ |
| 住所解決 | 事業所会員: 勤務先* / 個人・賛助: 郵送先区分コード（OFFICE/HOME）に従い勤務先* or 自宅* |
| 都道府県 | '大阪府' は出力しない。他府県のみ表示 |
| 住所不備 | 郵便番号・市区町村・番地のいずれかが空の場合は '住所不備' シートへ分類 |
| xlsx 生成 | 一時 SS → UrlFetchApp Bearer token export → base64 → 一時 SS 削除 |

出力シート構成:

| シート名 | 列 |
|---|---|
| 事業所会員 | 名前, 郵便番号, 住所, 建物名 |
| 個人会員 | 名前, 郵便番号, 住所, 建物名, 勤務先名 |
| 賛助会員 | 名前, 郵便番号, 住所, 建物名, 勤務先名 |
| 住所不備 | 名前, 会員種別, 住所不備の項目 |

**`_fillMailingSheet_(sheet, headers, rows)`** — 新規追加

- ヘッダー行を太字・グレー背景で書き込み
- データ行を追加し列幅を自動調整

**action routing** — 追加

if (action === 'generateMailingListExcel') {

}

## 設計上の決定

- **一時ファイル削除**: `try/finally` で確実に削除。GAS の 6 分制限内に完結する。
- **都道府県省略**: `prefecture !== '大阪府'` の条件で住所欄に連結。列分割ではなく住所文字列に含める。
- **退会予定の扱い**: 発送対象として含め、ステータス列は出力しない。
- **事業所会員の名前**: `姓 + 名`（代表者名）を使用。勤務先名は住所解決に使うが列としては不要（事業所単位の宛先のため）。
