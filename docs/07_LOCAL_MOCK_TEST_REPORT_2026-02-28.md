- URL: `http://127.0.0.1:5173`
- 実行方式: Viteローカル起動 + Playwright操作
- 対象ブランチ: 作業中ワークツリー

## 2. テスト項目
1. 会員ステータスにログインIDが分かりやすく表示されること
2. 年会費が未納の会員のみ振込先口座情報が表示されること
3. パスワード変更フォームが動作すること
4. 受付中研修で「詳細表示」ができること
5. 受付中研修で「案内PDFリンク」が表示されること
6. 管理者ログイン時のみ左メニューに「管理者ページ」が表示されること
7. 管理者ログイン時も会員情報が紐づいた状態で表示されること

- 本件の必須要件（会員ステータス/会費未納表示/パスワード変更/研修詳細+PDF/管理者表示）は満たす
