# リリース記録 v249（2026-04-20）

## 概要

会員マイページと管理者ポータルの UI を完全分離した。
会員ポータル URL にアクセスしても管理者ログインタブが表示されなくなる。

## 修正内容

### scripts/build-gas.mjs

- `VITE_APP=member` でビルド → `backend/index.html`（会員専用、管理者タブ非表示）
- `VITE_APP=admin` でビルド → `backend/index_admin.html`（管理者専用）を新規追加
- v248 以前は `VITE_APP` 未設定（統合モード）のため会員ページにも管理者タブが表示されていた

### backend/Code.gs（doGet）

- `allowedApps` に `'admin': 'index_admin'` を追加
- `APP_TITLES` に `admin: '管理者ポータル｜枚方市ケアマネ協議会'` を追加
- アクセス: `?app=admin` で管理者専用ページを返却

## URL 体系（確定）

| 用途 | URL |
|---|---|
| 会員マイページ | `https://script.google.com/.../exec`（引数なし） |
| 管理者ポータル | `https://script.google.com/.../exec?app=admin` |
| 公開ポータル | `https://script.google.com/.../exec?app=public` |

※ 会員用 deployment ID と管理者・公開は同じ deployment を共有（URL パラメータで分岐）

## Release 手順

1. `npm run typecheck` ✅
2. `npm run build:gas` ✅（5 files: member/public/admin + compress）
3. `npx clasp push --force` ✅（5 files pushed: index_admin.html 新規追加）
4. `npx clasp version "v249 ..."` ✅（version 249 created）
5. `npx clasp redeploy <member-id> --versionNumber 249` ✅
6. `npx clasp redeploy <public-id> --versionNumber 249` ✅

## 操作者確認ポイント

1. **会員マイページ**: 会員URL にアクセス → 「会員ログイン」タブのみ表示（管理者タブなし）
2. **管理者ポータル**: `?app=admin` でアクセス → 「管理者ログイン」タブのみ表示
3. **公開ポータル**: `?app=public` でアクセス → 公開研修一覧が表示
